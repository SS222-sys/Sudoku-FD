Слой / Файл	Назначение	Использует	Примечания
main.py	Точка входа SPA	app_layout.py, services, config.py	Инициализирует зависимости и запускает приложение

config.py	Глобальные настройки	–	Размеры окна, уровни сложности, параметры по умолчанию

styles	Темы оформления	–	Используются только UI-компонентами
components	UI-компоненты	styles, assets, utils	Переиспользуемые элементы интерфейса

views	Экранные панели / страницы	components, services, styles, assets, utils	Отображение экранов и обработка пользовательских событий

app_layout.py	Каркас	views, components	Управляет переключением экранов без перезагрузки

services	Бизнес-логика и авторизация	database, utils	Панели вызывают сервисы; сервисы работают с БД

database	Доступ к базе данных	–	Хранение пользователей, прогресса и статистики

utils/helpers.py	Вспомогательные функции	–	Используется всеми слоями по необходимости
